# EEGTrainer
Harvard Makeathon 2021: Train your focus using an EEG with Machine Learning.

Use an EEG to measure how strong you are focusing.

Inspired by MIT research:
https://mitsar-eeg.com/2020/04/01/adhd_neurofeedback-2/
